# Nike, Inc. — FY2024 Sales Performance Analysis

An independent data analysis of Nike's fiscal year 2024 financial performance, built using Python and presented as an interactive dashboard.

## What's In This Repo

| File | Description |
|---|---|
| `index.html` | Interactive dashboard — charts, findings, and recommendations |
| `Nike_FY2024_Sales_Analysis.ipynb` | Python notebook — full analysis using Pandas and Plotly |

## Live Dashboard
👉 [View Interactive Dashboard](https://yonisv.github.io/nike-sales-analysis/)

## What I Analyzed
- Revenue by geographic segment (North America, EMEA, Greater China, APLA)
- Revenue by product category (Footwear, Apparel, Equipment)
- Wholesale vs. Direct-to-Consumer channel performance
- Year-over-year comparisons: FY2023 vs FY2024

## Key Findings
- Total revenue grew just 0.3% — essentially flat on a $51.4B base
- North America (-1.0%) is the only declining region
- APLA (+4.6%) and Greater China (+4.1%) are the growth engines
- DTC now represents 43.5% of Nike Brand revenue — up from ~30% five years ago
- Footwear at 68.6% of revenue represents a significant concentration risk

## Tools Used
- Python · Pandas · Plotly · Chart.js · HTML/CSS

## Data Source
Nike, Inc. Form 10-K FY2024 — SEC EDGAR · Fiscal Year End: May 31, 2024
